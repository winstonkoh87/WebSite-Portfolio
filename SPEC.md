## 2. Technical Stack

- **Host**: GitHub Pages. **[NO VERCEL]**
- **Core**: Vanilla HTML5, CSS3, ES6.
- **Visuals**: Particles.js (low density), JetBrains Mono, dark theme.
- **Diagrams**: Mermaid.js with custom teal theme.
- **Accessibility**: `prefers-reduced-motion` respected.

## 3. Architecture

- **Root**: `index.html`, `athena.html`, `portfolio.html`.
- **`projects/`**: Self-contained sub-sites (Brew & Bean, Melvin).
- **`assets/`**: Shared Design System (style.css, terminal.css, particles.js, typewriter.js).

## 4. Design Principles (V2.2)

1. **Builder Narrative**: Show the system running, not just the blueprint.
2. **Verifiable Artifacts**: Metrics must be real (memories indexed, query time, protocols).
3. **Anti-Hype**: Calm, specific, technical. No marketing hyperbole.
4. **Local-First Positioning**: Athena is framed as local-first, privacy-respecting.
